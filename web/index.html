Survey Repository is a small service that allows the centralisation of survey definitions for the InfluenzaNet project. The aim of this project is to improve the documentation of the collected data and to facilitate the comparison of data between participating countries. This repository contains only the definitions of the surveys (json) and it's not intentented to receive any collected data.
